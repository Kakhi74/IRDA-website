Bienvenue sur la page web de Dr.Khiari, veuillez déposer vos fichier concernant l'IRDA selon le format approprier.

Inserer le fichier de poids ici selon ce format ---> "Poids [directory].xlsx"

Example ---> Poids HNO3_HCL.xlsx

Inserer le fichier des résultats de l'IRDA ici selon ce format ---> "Résultats_IRDA [directory].xlsx"

Example ---> Résultats_IRDA HNO3_HCL.xlsx

Aussurez vous que vos fichier de poids correspondent aux fichier de résultats IRDA et que ceux-ci ont été soumis avec le nom de fichier approprier
